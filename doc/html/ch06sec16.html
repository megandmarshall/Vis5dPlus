Tcl scripting

Vis5D 5.2 features a scripting facility. That is, you can control Vis5D with a text file of commands using the Tcl language. Scripting is an advanced subject and is documented separately in the Vis5D scripting document.

Note that the SAVE and RESTORE buttons on the control panel write and read Tcl files. You may want to use bits of these files as a basis a new Tcl scripts.
