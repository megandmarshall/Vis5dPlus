Vis5D Documentation on the Web

The Vis5D Home Page and Vis5d+ Home Page are available on the World Wide Web. You can also find this manual online.

The Vis5d page links to another Web document that describes how to use Vis5D files as a World Wide Web medium for exchanging model output. A second document describes the Vis5D API (Application Programmer Interface), intended to help system developers use Vis5D as a visualization subsystem of other systems. Finally, there is a document describing the Vis5D Tcl scripting interface.
